------------------------------------------structure du site------------------------------------------

timemachine / source file
	
	1. DOSSIER src/
		
		1.1 DOSSIER Controller/ 
			# Contient les fichiers qui gèrent les actions ( = Class) sur le site
			admin/ --> idem coté admin
				

					MembresController.php
					PhotosController.php
					StoriesController.php			
				
			MembresController.php
			PhotosController.php
			StoriesController.php		
		
		1.2 DOSSIER Entity/ --> correspond aux classes des objets que l'ont va créer: membre/photo/stories
						chaque élément sera un objet d'une de ces classes
				--> ce dossier va aussi correspondre aux namespaces
				membre.php
				photo.php
				story.php
			membre.php
			photo.php
			stories.php		
		
		1.2 DOSSIER Repository/ --> contiendra les requêtes BDD
			1.2.1 FICHIER MembresRepository.php
			1.2.2 FICHIER PhotosRepository.php
			StoriesRepository.php	
		
		1.3 FICHIER app.php :
			# tous les services seront déclarés dans ce fichier
				-> déclarer les CONTROLLER 
				-> déclarer les REPOSITORY
				-> déclarer les AUTRES SERVICES (password)
					=> exemple de la homepage
						/* Homepage */
						$app['index.controller'] = function() use ($app){ // use pour éviter global $app
							//  global $app;
							return new \Controller\IndexController($app);
						};

		1.4 FICHIER controllers.php
			# toutes les routes seront déclarées dans ce fichier
				-> FRONT
				-> BACK
				-> RUBRIQUES
				-> ARTICLES
					=> exemple
						$app
							->get('/', 'index.controller:indexAction')
							->bind('homepage')
						;

	template/ --> les vues
		admin/ --> vues du back-office
		autres_vues.html.twig (plusieurs fichiers)






------------------------------------------fonctionnalités du site------------------------------------------

fonctionalités visiteur (non connecté) :

	- s'inscrire
	- consultation du site
	- signalement des contenus inappropriés


fonctionalités membre :

	- se connecter
	- consultation du site
	- signalement des contenus inappropriés
	- upload des photos
	- noter une photo
	- commenter une photo
	- statistiques utilisateur
	- créer stories / dossier thématique
	- se désinscrire
	- modifier son profil


fonctionalités admin :

	 - accès statistiques du site
	 - changer status user
	 - (modifier une photo)
	 - supprimer une photo
	 - supprimer un membre
	 - changer le mot de passe d'un membre
	 - notification des contenus inappropriés
 
 
fonctions de recherche :

- le site aura des filtres de recherches classiques:
	- par lieux
	- par date
	- par catégorie
	- par mot clés
	
- le site aura des outils de recherche modernes (ca ne va peut-être pas être simple)
	- timeline avec curseur interactif qui affiche l'année sélectionnée
	- carte du monde interractive pour la sélection de la région


- fonctionnalités supllémentaires :

	- liens vers autres réseaux sociaux
	- APIs
